# Changelog

## [2.0.0] - 2025-12-30

### ADDED
- **AI Assistant:** Full conversational interface for kitchen help.
- **Multi-model Analysis:** Intelligent food detection using Gemini and other vision models.
- **Smart Notifications:** Personalized hydration and nutrition reminders.
- **Memory-based Suggestions:** AI-driven meal preferences based on history.

- **Premium UI Overhaul:** Dark mode optimization and refined design system.
- **Security:** Encrypted local database and secure credential storage.
- **Performance:** Image compression and response caching.
- **Analytics:** Comprehensive tracking with Firebase and Sentry.
- **Offline Sync:** Enhanced Firestore persistence for unreliable networks.
